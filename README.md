# AimLock
AimLock — это оффлайн 2D-шутер, в котором ты сражаешься против бесконечных волн врагов. Управляй персонажем с помощью WASD, целись мышью и расстреливай врагов, пока они не добрались до тебя.

# Как играть?
• Скачай Python
• Запусти: python [версия игры].py
• Играйте

# Управление:
• WASD — движение
• Мышь — прицеливание
• ЛКМ — стрельба
• ESC — выход
